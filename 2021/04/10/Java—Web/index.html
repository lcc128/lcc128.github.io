Html

align 居中对齐

参数 描述
align =”left” 左对齐内容。
align =”right” 右对齐内容。
align =”center” 居中对齐内容。

form表单

属性 描述
accept-charset 规定在被提交表单中使用的字符集（默认：页面字符集）。
action 规定向何处提交表单的地址（URL）（提交页面）。
autocomplete 规定浏览器应该自动完成表单（默认：开启）。
enctype 规定被提交数据的编码（默认：url-encoded）。
method 规定在提交表单时所用的 HTTP 方法（默认：GET）。
name 规定识别表单的名称（对于 DOM 使用：document.forms.name）。
novalidate 规定浏览器不验证表单。
target 规定 action 属性中地址的目标（默认：_self）

input

属性 参数 描述
placeholder text 规定帮助用户填写输入字段的提示。
accept mime_type 规定通过文件上传来提交的文件的类型。
size number_of_char 定义输入字段的宽度。
value value 规定 input 元素的值。
autocomplete on off 规定是否使用输入字段的自动完成功能。

button

方法 描述
blur() 把焦点从元素上移开。
click() 在某个按钮上模拟一次鼠标单击。
focus() 为某个按钮赋予焦点。

超链接

CSS 指的是层叠样式表,描述了如何在屏幕、纸张或其他媒体上显示 HTML 元素,节省了大量工作。它可以同时控制多张网页的布局

CSS语法

常用参数

颜色背景设置

参数 作用
color 设置字体颜色
background-color 设置背景颜色
opacity：0.0-1.0 设置不透明度
rgba(red, green, blue, alpha) 设置指定颜色的不透明度

图片背景

参数 作用
background-image: url(“/i/paper.jpg”) 设置指定图片为背景
backfround-size：20px 指定背景图片的大小
background-repeat: repeat-x; 设置图片的方向
background-repeat: no-repeat; 设置图片背景只显示一次
background-position: right top; 设置图片背景的位置
background-attachment: fixed; 指定图片背景的位置是固定的

边框

border-style 属性指定要显示的边框类型

  • dotted - 定义点线边框
  • dashed - 定义虚线边框
  • solid - 定义实线边框
  • double - 定义双边框
  • groove - 定义 3D 坡口边框。效果取决于 border-color 值
  • ridge - 定义 3D 脊线边框。效果取决于 border-color 值
  • inset - 定义 3D inset 边框。效果取决于 border-color 值
  • outset - 定义 3D outset 边框。效果取决于 border-color 值
  • none - 定义无边框
  • hidden - 定义隐藏边框

  • 使用 document.write() 写入 HTML 输出
  • 使用 innerHTML 写入 HTML 元素
  • 使用 console.log() 写入浏览器控制台
